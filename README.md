 - cards repartition
 - chance to get a legendary/rare/common on packs

## Setup: 
Install angular cli
```bash
npm install -g angular-cli
```
Install the dependencies : 
```bash
npm install
```
Serve the app : 
```bash
ng serve
```
The app is now available on http://localhost:4200/
